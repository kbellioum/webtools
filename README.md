# webtools

This tools help us doing some tests like :

 - Ping
 - Whois
 - Port Scan

I'm working on other tools


Installation steps :
--------------------


```bash
get clone https://github.com/kbellioum/webtools.git
cd webtools
npm install
```
to execute :
```bash
npm start
```
